Edit draft test case

The following guidance shows you how to edit a draft test case in Katalon Cloud Studio (Beta).

Edit test case description​

Learn how to edit the description of a draft test case in Katalon Cloud Studio (Beta).

  1. From the Project home screen, go to Tests > Test Cases and select the draft test case you wish to edit.
  2. Hover and click the Description field to change the field in Edit mode, and update your draft test case description.
  3. Click Save to save your test case description.
Your test case description is now updated.

Add a new test step​

Learn how to add test step(s) on your draft test case in Katalon Cloud Studio (Beta).

There are two ways to add a new test step in a draft test case in Katalon Cloud Studio (Beta):

  • Use the Add new test step button;

  • Use the Plus (+) icon between test steps.

  1. From the Project home screen, go to Tests > Test Cases and select the draft test case you wish to edit.
    Note: If you are creating a new Katalon Cloud Studio (Beta) test case, follow the steps in the topic Create a draft test case and start with the next step (Step 2).
  2. Add new test step(s) using any of the following ways:
    1. Click the + Add New Test Step button at the bottom of the test steps list.
    2. Hover between two test steps and click the Plus (+) icon to add a new test step in between.
  3. Select any of the Supported actions you wish to add as a new test step.
    If the selected action requires an object, you can select the object to perform the action on from the object list or capture a new object by clicking Capture new object. Fill in any other inputs (ex. text, URL, password) as needed. For more information about this step, see: Supported actions.
  4. Click Save to save your changes.
Your test steps have been added to your test case.

Cut and copy test steps​

You can cut and paste or copy and paste) selected test step(s) in your draft test case.

  1. From the Project home screen, go to Tests > Test Cases and select the draft test case you wish to edit.
  2. Select the test steps you wish to cut or copy using any of the following ways:
    1. Hover along the test step to the right and click the overflow (3-dot) menu and select Cut or Copy:
    2. If you select multiple steps, right-click and select Cut or Copy all selected test steps:
    3. You can also click on the toolbar icons to either cut or copy your selected test steps.
  3. Place your cursor where you like to paste the cut or copied steps, then right-click and click Paste. In the example below, the cursor is placed on the empty space area at the end of the test steps to paste them as new test steps.
    Note:
    • If you copied a test step, you can paste the test step multiple times.
    • If you cut a test step, you can only paste it one time only.
The cut/copied test steps have been successfully pasted in the location where you wish to add them.

Click Undo on the toast notification to undo the cut and paste or copy and paste action.

Learn how to move and organize test steps in a draft test case using drag and drop in Katalon Cloud Studio (Beta).

Here is a quick demo on how to move test steps via drag and drop:
Note: To activate this action, hover on the selected test step to make the Drag icon appear.

    1. In the Editor view, hover along the test step to the right till you see the Duplicate icon. Click the Duplicate icon to duplicate the test step:

Bulk-duplicate test steps​

Learn how to bulk-duplicate test steps of a Katalon Cloud Studio (Beta) test case.
Note: To activate inline and toolbar actions, hover on the selected test step.
  1. From the Project home screen, go to Tests > Test Cases and select the draft test case you wish to edit.
  2. In the Editor view, click the checkboxes associated to the test steps. You can also use the Shift + Arrow Down or Up keys on your keyboard to select the test steps.
  3. You can bulk duplicate selected test steps using any of the following ways:
    1. Click the Duplicate icon on the toolbar.
    2. Right-click and select Duplicate all test steps.

Bulk-delete test steps​

Learn how to bulk-delete multiple test steps in a Katalon Cloud Studio (Beta) test case.

  1. From the Project home screen, go to Tests > Test Cases and select the draft test case you wish to edit.
  2. In the Editor view, click the checkboxes associated to the test steps. You can also use the Shift + Arrow Down or Up keys on your keyboard to select the test steps.
  3. You can bulk delete test steps using any of the following ways:
    1. Click the Delete icon on the toolbar.
    2. Right-click and select Delete all test steps.
The selected test steps of your draft test case have been deleted.

Click Undo on the toast notification to undo and recover the recently deleted test steps.

Edit test step​

Learn how to edit your test step in Katalon Cloud Studio (Beta).

  1. From the Project home screen, go to Tests > Test Cases and select the draft test case you wish to edit.
  2. In the Editor view, hover on the test step you wish to edit and click the Edit icon.
  3. Depending on the action you selected as test step, the Editor automatically renders the input and/or object for you. You may edit the input and/or select the object to perform the action on. For more information about which action is supported to edit your test case, see: Supported actions.
    If the object is not in the dropdown list of captured objects, you can capture a new one using the Katalon Platform Recorder extension. For more information, see: Capture new object.
  4. Click Save to save your changes.
Your test step is edited.

Capture new object​

Learn how to capture a new object using Katalon Platform Recorder.

  • You need to have Katalon Platform Recorder extension installed. For more details, see: Download and install Katalon Platform Recorder.
Katalon Platform Recorder allows you to easily and quickly capture a new object while editing or adding a new test step in your Katalon Cloud Studio (Beta) draft test case.
  1. From the Project home screen, go to Tests > Test Cases and select the draft test case you wish to edit or add a new test step.
  2. In the Editor view, hover on the test step you wish to edit and click the Edit icon or proceed with the steps to Add a new test step.
  3. On the object field, click the drop-down and select Capture New Object.
    Katalon Cloud Studio (Beta) then launches the AUT and navigates the AUT to the nearest URL of the current editing step.
  4. Click the object you want to capture, then navigate to the floating menu and click Save to finish capturing the object.
The newly-captured object is displayed and automatically selected as the object for the current editing step.
